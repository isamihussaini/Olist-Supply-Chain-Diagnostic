# Executive Supply Chain Dashboards

This folder contains the final Power BI deliverables designed for stakeholders to track logistics risk.

### Dashboard Features:
* **Risk Diagnostic Page:** High-level view of the $1.35M revenue-at-risk.
* **Operational Trends:** Time-series analysis identifying the March 2018 systemic collapse.
* **Regional Risk Map:** Drill-down capability to isolate specific cities (Maceió, São Gonçalo) with high bottleneck frequency.

**Files Included:**
* `.pbix` file for interactive exploration.
* `.pdf` export for quick executive review.
